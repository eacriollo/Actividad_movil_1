# Actividad 01 – Aplicación Android

**Programación en dispositivos móviles**

## Descripción del proyecto

Este proyecto corresponde al desarrollo de una aplicación móvil nativa para Android, cuyo objetivo es 
aplicar los conocimientos adquiridos en el desarrollo de aplicaciones utilizando Kotlin y Jetpack Compose.

La aplicación, denominada **“Mundo Anime”**, permite al usuario explorar personajes de dos universos temáticos: 
Pokémon y Dragon Ball. A través de una interfaz sencilla, el usuario puede navegar entre categorías, visualizar 
listados de personajes, realizar búsquedas y acceder al detalle de cada uno.

---

## Objetivos

* Aplicar el uso del lenguaje Kotlin en el desarrollo Android.
* Implementar interfaces modernas utilizando Jetpack Compose.
* Utilizar arquitectura MVVM para estructurar la aplicación.
* Consumir datos desde APIs REST externas.
* Fomentar la reutilización de componentes y buenas prácticas de desarrollo.

---

## Características principales

* Navegación entre múltiples pantallas.
* Listado de personajes por categoría.
* Búsqueda de personajes.
* Visualización de detalles.
* Consumo de APIs públicas.
* Uso de componentes reutilizables.
* Gestión de estados en la interfaz.

---

## Arquitectura

El proyecto sigue el patrón **MVVM (Model-View-ViewModel)**, separando claramente:

* Capa de presentación (UI con Compose)
* Lógica de negocio (ViewModel)
* Acceso a datos (Repository / API)

---

## Tecnologías utilizadas

* Kotlin
* Jetpack Compose
* Retrofit (librería)
